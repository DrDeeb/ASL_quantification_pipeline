# ASL quantification pipeline using Neurolens 2.094

This pipeline required:
- Neurolens 2 (>=2.094)
  - Motion Correction plugin
  - ASL Quantification plugin
  - GLM Time Series plugin
  - Spatial Filtering plugin
  - ROI Averaging plugin

- MINC tool
  - mnc2nii
  - nii2mnc
  - mincstats
  - mritoself
  - mincmath
  - mincresample
  - minc_modify_header
  - mincheader

- CIVET (for GM mask)

- FSL
  - bet
  - fslmaths
  - fugue
  - prelude

- XML simple
